Display Components

label, progressBar, energyBar, fluidBar.

Display components render information. They do not accept user input.

label

ui.label(Component.literal("Hello"))
ui.label(Component.literal("")).bind("status")  // dynamic

progressBar

ui.progressBar()
    .bind("progress")   // float 0.0–1.0
    .fillWidth().height(10)

energyBar

ui.energyBar()
    .bind("energy", "maxEnergy")
    .size(12, 54)

Renders vertically by default. Tooltip shows current / max values.

fluidBar

ui.fluidBar()
    .bind("fluid", "fluidMax")
    .size(12, 54)

Same as energyBar but renders the fluid texture and color.
