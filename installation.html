Installation

Adding UIWeaver to your mod project.

Requirements

RequirementVersion
Minecraft1.20.1
Forge47.x
Java17+

Adding UIWeaver as a dependency

In your build.gradle:

repositories {
    maven { url "https://maven.pkg.github.com/BgameXL/UIWeaver" }
}

dependencies {
    implementation fg.deobf("com.bgame:uiweaver-forge:0.1.0")
}

In gradle.properties:

uiweaver_version=0.1.0

Verify the setup

After syncing Gradle, you should have access to the com.bgame.uiweaver package. Import UIWeaver in any class to confirm it resolved correctly:

import com.bgame.uiweaver.UIWeaver;
Note: UIWeaver is licensed under LGPL-3.0. You can use it as a library in closed-source mods as long as modifications to UIWeaver itself stay under LGPL-3.0.
