Component API

Method reference for all components.

Quick reference for all common methods available on UIWeaver components.

Universal

MethodDescription
.id(String)Assign an ID for binding, inspection, and targeting
.size(w, h)Fixed pixel size
.width(w)Fixed width
.height(h)Fixed height
.fillWidth()Expand to parent's available width
.fillHeight()Expand to parent's available height
.grow()Take remaining space along parent axis
.visible(bool)Show or hide (hidden components still occupy space)
.tooltip(Component)Tooltip shown on hover

Containers only

MethodDescription
.padding(n)Inner padding all sides
.padding(x, y)Horizontal and vertical padding
.gap(n)Space between children
.add(component)Add a child component

Bindable components

MethodDescription
.bind(key)Bind to a single ViewModel value
.bind(key1, key2)Bind to two values (e.g. current + max)

Interactive components

MethodDescription
.action(String)Named action sent to server on interaction
.value(n)Initial value (slider)
.placeholder(String)Placeholder text (textInput)

UIWeaver.screen()

MethodDescription
.root(component)Set the root component
.viewModel(vm)Attach a ViewModel
.open()Send to player and open the screen
