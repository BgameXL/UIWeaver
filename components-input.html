Input Components

button, toggle, checkbox, slider, textInput.

Input components send server actions or toggle local display state.

button

ui.button(Component.literal("Start"))
    .id("btnStart")
    .action("start")
    .size(60, 20)

toggle

ui.toggle()
    .id("activeToggle")
    .action("setActive")
    .bind("active")

Displays as an on/off switch. Sends action with current boolean value.

checkbox

ui.checkbox(Component.literal("Enable filter"))
    .action("setFilter")

slider

ui.slider(0, 100)
    .id("speed")
    .value(50)
    .action("setSpeed")
    .width(120)

Sends the integer value with the action when released.

textInput

ui.textInput()
    .id("nameField")
    .placeholder("Enter name...")
    .action("setName")
    .width(140)

Sends the string value with the action on Enter or focus loss.
