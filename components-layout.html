Layout Components

column, row, grid, tabs, scrollPanel.

Layout components hold and position other components. They do not render any visible content themselves.

column

ui.column()
    .padding(8).gap(6)
    .add(child1).add(child2)

Stacks children top to bottom. Default width is the widest child. Default height is the sum of children plus gaps.

row

ui.row()
    .gap(8).fillWidth()
    .add(child1).add(child2)

Places children left to right. Default height is the tallest child.

grid

ui.grid(cols, rows)
    .gap(4)
    .add(child1) ... .add(childN)

Fills cells left to right, top to bottom. All cells are equal size.

tabs

ui.tabs()
    .tab("Label", panel)
    .tab("Label2", panel2)

Renders a tab bar above the active panel. Only the active panel participates in layout and input.

scrollPanel

ui.scrollPanel()
    .size(200, 140)
    .add(ui.column().gap(4).add(...))

Clips its content to the panel bounds and allows scrolling. Supports mouse wheel.
